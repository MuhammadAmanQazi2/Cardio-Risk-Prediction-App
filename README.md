Cardio Risk Prediction App 🫀
A real-time web application built with Streamlit that leverages Machine Learning models to assess the risk of cardiovascular disease based on patient health metrics. This project demonstrates a complete end-to-end data science workflow—from data preprocessing and model training to deployment.

🚀 Overview
Heart disease remains one of the leading causes of mortality worldwide. This application aims to provide an accessible tool for early risk assessment. Users can input clinical parameters (such as age, BMI, blood pressure, and smoking status) to receive an immediate prediction of their cardiovascular health risk.

🛠️ Tech Stack
Language: Python

Machine Learning: XGBoost, Random Forest, Scikit-learn

Web Framework: Streamlit

Data Manipulation: Pandas, NumPy

Visualization: Matplotlib, Seaborn

✨ Key Features
Dual Model Approach: Utilizes both Random Forest and XGBoost to ensure robust and accurate predictions.

Real-time Prediction: Instant feedback based on user input via a clean, intuitive UI.

Data Augmentation: Techniques applied during training to handle class imbalance and improve model precision.

Feature Importance: Focused on key clinical indicators like BMI, glucose levels, and hypertension.

📊 Methodology
Exploratory Data Analysis (EDA): Performed deep analysis to identify correlations between lifestyle factors and heart disease.

Preprocessing: Handled missing values, normalized numerical features (like BMI and age), and encoded categorical data.

Model Selection: Evaluated several classifiers; XGBoost and Random Forest were chosen for their high precision and recall scores in medical contexts.
